Backend :
menggunkana framework express dengan mongodb sebagai database dan mongoose sebagai data modelingnya.
JWT untuk authentikasinya.
untuk menjalankan dengan ketik "npm run start"

Frontend:
React sebagai frameworknya.
React-cookie untuk menyimpan coockie.
React-router-dom untuk menangani routernya.
Axion untuk menghubungkan ke backendnya.
untuk menjalankan dengan ketik "npm run dev
